# Power Smart Classrooms

Power Smart Classrooms is an AI-powered classroom monitoring system that uses computer vision to detect and analyze classroom activities in real time. The project leverages YOLO (You Only Look Once) object detection to identify students and classroom events, helping improve classroom management and monitoring.

## Features

- Real-time object detection using YOLO
- Live video processing with OpenCV
- Classroom activity monitoring
- AI-powered analytics and insights
- Interactive web interface using Streamlit

## Technologies Used

- Python
- Streamlit
- OpenCV
- YOLO (Ultralytics)
- NumPy
